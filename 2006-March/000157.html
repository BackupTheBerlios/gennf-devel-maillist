  Log:
  1. Added a function and a macro to work with temporary files.
     Those are available in osicat in principle but they do not return
     the name of the temporary file but only an open stream. As
     it is sometimes necessary to pass the name of the temporary file
     to another program or routine it is returned as second value or
     bound to an optional argument.
  2. in-temporary-directory and in-directory now restore the old state on
     non-local exits. in-temporary-directory allows to disable this behaviour.
