  Log:
  I did so many changes that I don't know all of them anymore.
  Basically, gennf is somewhat operable now. Merging is half-way implemented,
  branching, checkout, commit, update seem to work. The handling
  of the directory structure has still some quirks but I have introduced
  a *sandbox-directory* and a *startup-directory* variable to resolve
  filenames against. The map file now only stores string, no pathnames.
  Furthermore, the identification must not contain a META/n with n being
  a branch number because that makes merging much harder.
  There is now a hook to plug in customized merge routines for files.
  This is used to overcome an initial difficulty when merging
  a NIL map file againts some sequence of mappings. But it can as
  well be improved a lot for better merging, which is very poor at the moment.
